## Installation Steps
- 1.Download the correct ZIP
[release](https://github.com/RSE-E/Better_Hats_Pack/releases) based on your Modding Us version.
- 2.Open Modding Us.
- 3.Go to "mods" tab.
- 4.Click "import" button.
- 5.Select mods zip file.
- 6.Press "quick restart" button.
